# 副作用

## 副作用操作

1. 调用接口
2. 修改 DOM(操作元素样式，位置，焦点等)
3. 设置定时器（setInterval,setTimeout)
4. 订阅 WebSocket , 事件（如监听滚动，广播，键盘输入）
5. 操作浏览器 API(localStorage,sessionStorage,cookies)
6. 改变 URL (window.location)

## 如何在 React 实现副作用？

```js
useEffect(() => {
  // 副作用逻辑， 比如：请求数据，订阅，修改DOM
  return () => {
    // 清理副作用，比如清除定时器，取消订阅
  };
}, [依赖]);
```

1. 空数组，副作用函数只在组件挂载时执行一次（componentDidMount）
2. 依赖项，副作用函数在依赖项变化时重新执行（componentDidUpdate，指定条件更新）
3. 没有依赖数组，每次渲染都会执行副作用函数 (componentDidUpdate, 但啥都监听)

- 复杂副作用，建议拆分多个 useEffect
